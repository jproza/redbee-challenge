1) Clonar el repo -> https://github.com/jproza/redbee-challenge.git
2) Importar el proyecto desde eclipse (existing maven project)
3) Agregar usuario y constraseña en la clase que contiene la signatura main (runneable).
4) En apps.twitter.com (logueado en twitter) crear una aplicacion twitter y definir :  
Consumer Key (API Key)	
Consumer Secret (API Secret)
Access Token
Access Token Secret
5) Agregar las claves del paso 4 en el archivo "twitter4j.properties"
6) Ejecutar la clase que contiene el metodo main.
